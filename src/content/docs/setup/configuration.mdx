## UnoCSS 配置

文件: `uno.config.ts`

这是一个实用优先的 CSS 框架，用于构建自定义设计。主题还使用其预设 `typography` 进行排版设置（查看[其他集成#`@unocss/preset-typography`](/docs/integrations/others#unocsspreset-typography) 了解更多信息）。

了解更多：

- [UnoCSS: 排版预设](https://unocss.dev/presets/typography)
- [UnoCSS: Astro 集成](https://unocss.dev/integrations/astro)
- [UnoCSS: 配置 UnoCSS](https://unocss.dev/config)

您还可以通过修改位于 `src/assets/styles/app.css` 的 CSS 文件来自定义主题默认的 UnoCSS 调色板。例如，如果您想更改默认主题颜色，可以修改以下代码：

```css title="src/assets/styles/app.css"
:root {
  /* ... */
  --primary: 200 29% 45%; /* [!code --] */
  --primary: <您喜欢的颜色，使用原始 hsl 值>; /* [!code ++] */
}
```
